# STM32 + TouchGFX 嵌入式踩地雷遊戲實作
本專案以 STM32 系列 MCU 為核心平台，結合 TouchGFX 圖形化介面框架，實作嵌入式踩地雷遊戲應用系統。專案目的在於整合 MCU 韌體控制與 GUI 顯示邏輯，驗證嵌入式系統在人機互動應用上的完整開發流程。

系統透過 TouchGFX 建構遊戲介面與操作邏輯，包含格子生成、點擊事件判斷、地雷分佈演算法與勝負條件控制。底層由 MCU 負責事件處理與狀態管理，並透過中斷機制與任務流程控制確保操作反應流暢。

開發過程中重點包含：

MCU 周邊設定與初始化（使用 STM32CubeMX 進行配置）

GUI 事件處理流程設計

遊戲狀態機邏輯實作

觸控輸入與顯示刷新整合

韌體與圖形框架資源配置管理

此專案強化了嵌入式系統中「軟硬體整合」與「事件驅動架構」的實作能力，並累積 GUI 應用於資源受限平台上的開發經驗。
